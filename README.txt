
The code needs "File" folder and "File1" folder to be in the same folder in which the code is

The code take the length of DNA strands as input, for example if the length of the first strand is 100 give 100 as input and if the length of the second strand is 200 give 200 as the next input

"File" folder contains a DNA sequences and "File1" folder also contains another DNA sequence

Paste the File and File1 in the same location where the program is saved. 

This website--> https://www.bioinformatics.org/sms2/random_dna.html can be used to generate random DNA sequence
